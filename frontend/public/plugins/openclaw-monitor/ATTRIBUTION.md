## Monitor Asset Attribution

This plugin keeps all vendored assets inside this folder and mixes two upstream sources:

### 1. Character sprites

Character sprites are derived from the official Stanford `generative_agents` repository:

- Repository: https://github.com/joonspk-research/generative_agents
- Repository license: Apache License 2.0

The upstream README separately credits the original artists for the game assets included in that research demo. Relevant attribution noted there includes:

- Background art: PixyMoon (@_PixyMoon_)
- Furniture / interior design: Mewyum
- Character design: Devin Nunez (@DevinNuNunez)

Only a small curated roster of character frames was extracted and repacked here for the TopicLab monitor.

### 2. Office scene assets

The office background, sofa, desk, plant, poster, and server-room assets are adapted from `Star-Office-UI`:

- Repository: https://github.com/ringhyacinth/Star-Office-UI
- Code / logic: MIT
- Art assets notice: non-commercial only, per the upstream `LICENSE`

This plugin uses a trimmed subset of the Star Office visual assets for learning / demo usage and keeps the original attribution requirement in mind.
