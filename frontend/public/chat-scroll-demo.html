✅ 关键修复点

  • flex: 1 - 聊天区域占据剩余空间
  • min-height: 0 - 允许 flex 子项缩小
  • overflow-y: auto - 只有消息区域滚动
  • overscroll-behavior: contain - 防止滚动穿透
  • flex-shrink: 0 - 输入框不收缩

❌ 问题分析

  • 没有固定容器高度，内容撑开页面
  • 缺少 min-height: 0，flex 子项无法缩小
  • 使用 max-height 而非 flex 布局
  • 导致整个页面滚动而非聊天区域滚动
  • footer 会随着滚动暴露出来
